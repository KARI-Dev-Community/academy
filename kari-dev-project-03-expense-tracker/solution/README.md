# Solution - Expense Tracker

This directory contains the complete reference implementation of the expense tracker.

## What's Included

- Full expense and income tracking
- Real-time balance updates
- Interactive Chart.js doughnut chart
- Transaction history with delete option
- Category filters
- localStorage persistence
- Responsive mobile design

## Tech Stack

- HTML5
- CSS3 (Grid, custom properties)
- JavaScript ES6+
- Chart.js for data visualization

## Running the Solution

```bash
git clone https://github.com/kari-dev/project-03-expense-tracker.git
cd project-03-expense-tracker/solution
npm install
npm run dev
```

Visit `http://localhost:3000` to see the completed project.
