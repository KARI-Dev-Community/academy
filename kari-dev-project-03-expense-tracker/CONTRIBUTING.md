# Contribution Guide - Project 03: Expense Tracker

Thanks for contributing to the expense tracker project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a Pull Request

## What You Can Improve

- Add income vs expense categorization
- Implement monthly budget limits with alerts
- Add export to CSV/PDF functionality
- Create pie and bar charts with Chart.js
- Support multiple currencies
- Add receipt image uploads
- Write data validation functions

## Pull Request Requirements

- Charts render correctly with sample data
- Calculations are accurate
- UI is responsive
- No broken links in documentation

## Getting Help

Open an issue with the `help-needed` label if you're stuck.
