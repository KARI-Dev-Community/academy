# Contribution Guide - Project 10: AI Study Assistant

Thanks for contributing to the AI study assistant project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a Pull Request

## What You Can Improve

- Add streaming responses for better UX
- Implement conversation history persistence
- Create study mode (quiz generation)
- Add PDF/document upload for context
- Implement RAG with ChromaDB or Pinecone
- Add multi-language support
- Create prompt templates library
- Add source citation in responses

## Pull Request Requirements

- AI responses are relevant and helpful
- API keys are not exposed in client code
- Conversation state handles edge cases
- UI shows loading/error states

## Getting Help

Open an issue with the `help-needed` label if you're stuck.
