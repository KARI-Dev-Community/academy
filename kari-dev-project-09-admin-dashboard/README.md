# Project 09: Admin Dashboard

A data-rich admin dashboard with charts and data tables.

## Learning Objectives

- Design data dashboard layouts
- Integrate charts and metrics
- Implement data filtering and export
- Access control for admin features

## Tech Stack

- React (Vite)
- Chart.js / Recharts
- CSS / TailwindCSS
- Node.js
- Express.js
- MongoDB

## Setup Guide

Frontend:

```bash
git clone https://github.com/kari-dev/project-09-admin-dashboard.git
cd project-09-admin-dashboard/frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
