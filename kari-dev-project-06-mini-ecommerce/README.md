# Project 06: Mini Ecommerce Store

A functional ecommerce storefront with product listings and cart.

## Learning Objectives

- Manage shopping cart state
- Handle product variants and quantities
- Implement checkout flow
- Design product listing UI

## Tech Stack

- React (Vite)
- CSS / TailwindCSS
- Node.js
- Express.js
- MongoDB

## Setup Guide

Frontend:

```bash
git clone https://github.com/kari-dev/project-06-mini-ecommerce.git
cd project-06-mini-ecommerce/frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
