# Contribution Guide - Project 01: Portfolio Website

Thanks for contributing to this portfolio project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a Pull Request

## What You Can Improve

- Add new sections (Projects, Blog, Contact form)
- Improve accessibility (ARIA labels, keyboard navigation)
- Add animations and transitions
- Optimize performance (lazy loading images)
- Write better CSS (CSS Grid layouts, custom properties)
- Add dark mode toggle

## Pull Request Requirements

- Code is clean and commented
- Works in latest Chrome, Firefox, and Safari
- Mobile responsive
- No console errors

## Getting Help

Open an issue with the `help-needed` label if you're stuck.
