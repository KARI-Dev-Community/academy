# Starter - Portfolio Website

This is your starting point for the portfolio project. The base structure is here; your job is to build it into a complete portfolio.

## What's Included

- Basic HTML5 boilerplate
- Simple CSS reset and base styles
- Placeholder sections for Home, About, and Projects
- Responsive meta tags

## What You Need to Build

### Required Features
1. Personal introduction in the About section
2. At least 3 project cards with descriptions
3. Contact form (can be static)
4. Smooth scrolling navigation
5. Mobile-friendly hamburger menu

### Stretch Goals
- Dark/light mode toggle
- Animated section reveals on scroll
- Resume download button
- Testimonials section

## Getting Started

```bash
git clone https://github.com/kari-dev/project-01-portfolio.git
cd project-01-portfolio
npm install
npm run dev
```

Open `http://localhost:3000` and start customizing!

## Tips

- Start with the HTML structure first
- Style each section one at a time
- Test on mobile early and often
- Use placeholder images from https://picsum.photos
