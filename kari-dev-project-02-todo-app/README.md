# Project 02: Todo App

A full-featured todo application with local storage persistence.

## Learning Objectives

- Manage application state
- Use browser localStorage
- Implement CRUD operations
- Build responsive UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API

## Setup Guide

```bash
git clone https://github.com/kari-dev/project-02-todo-app.git
cd project-02-todo-app
npm install
npm run dev
```
