# Contribution Guide - Project 02: Todo App

Thanks for contributing to the todo app project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a Pull Request

## What You Can Improve

- Add task categories and tags
- Implement due dates and reminders
- Add drag-and-drop reordering
- Create a dark mode theme
- Add task search and filtering
- Implement localStorage persistence improvements
- Write unit tests with Jest

## Pull Request Requirements

- All existing features still work
- New features are tested manually
- Code follows the existing style
- README is updated if needed

## Getting Help

Open an issue with the `help-needed` label if you're stuck.
