# Project 05: CRUD API

A RESTful API for managing a resource collection.

## Learning Objectives

- Design RESTful endpoints
- Implement CRUD operations
- Query and filter data efficiently
- Write API documentation

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- Postman (testing)

## Setup Guide

```bash
git clone https://github.com/kari-dev/project-05-crud-api.git
cd project-05-crud-api
npm install
cp .env.example .env
npm run dev
```
