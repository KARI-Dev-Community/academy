# Project Catalog

This directory explains each learning project and how to get started.

## Beginner Projects

### Project 01: Portfolio Website

Build a personal portfolio site to showcase your work and skills.

- **Difficulty**: Beginner
- **Tech**: HTML, CSS, JavaScript
- **Time**: 1-2 weeks
- **Key concepts**: Semantic HTML, responsive design, deployment
- [Go to repository](https://github.com/kari-dev/project-01-portfolio)

### Project 02: Todo App

Create a todo application with local storage persistence.

- **Difficulty**: Beginner
- **Tech**: HTML, CSS, JavaScript
- **Time**: 1-2 weeks
- **Key concepts**: DOM manipulation, state management, localStorage
- [Go to repository](https://github.com/kari-dev/project-02-todo-app)

### Project 03: Expense Tracker

Track income and expenses with Chart.js visualization.

- **Difficulty**: Beginner+
- **Tech**: HTML, CSS, JavaScript, Chart.js
- **Time**: 2-3 weeks
- **Key concepts**: Data visualization, calculations, filtering
- [Go to repository](https://github.com/kari-dev/project-03-expense-tracker)

## Intermediate Projects

- **Difficulty**: Intermediate
- **Tech**: Node.js, Express, MongoDB
- **Time**: 3-4 weeks
- **Key concepts**: JWT, password hashing, middleware, refresh tokens
- [Go to repository](https://github.com/kari-dev/project-04-jwt-auth-system)

### Project 05: CRUD API

Build a complete REST API for managing a resource collection.

- **Difficulty**: Intermediate
- **Tech**: Node.js, Express, MongoDB, Mongoose
- **Time**: 2-3 weeks
- **Key concepts**: REST, CRUD, validation, API testing
- [Go to repository](https://github.com/kari-dev/project-05-crud-api)

### Project 06: Mini Ecommerce Store

- **Tech**: React, TailwindCSS, Node.js, Express, MongoDB

## How to Choose Your First Project

1. Pick based on your current skill level
2. Complete projects in order unless you have prior experience
3. Read the starter guide before starting
4. Ask for help in the community if stuck
5. Build, break things, and learn
