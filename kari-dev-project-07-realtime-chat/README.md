# Project 07: Realtime Chat Application

A realtime chat app with multiple rooms and messaging.

## Learning Objectives

- Use WebSockets for realtime communication
- Manage chat rooms and participants
- Handle message broadcasting
- Build responsive chat UI

## Tech Stack

- React (Vite)
- Socket.io
- Node.js
- Express.js
- MongoDB

## Setup Guide

Frontend:

```bash
git clone https://github.com/kari-dev/project-07-realtime-chat.git
cd project-07-realtime-chat/frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
