# Contribution Guide - Project 07: Realtime Chat

Thanks for contributing to the realtime chat project!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a Pull Request

## What You Can Improve

- Add typing indicators
- Implement read receipts
- Create emoji picker
- Add file attachments support
- Build user presence indicators
- Add private messaging
- Implement chat history pagination
- Add message reactions

## Pull Request Requirements

- Messages appear instantly for all users
- Multiple rooms work independently
- UI handles empty states gracefully
- Connection/disconnection is handled

## Getting Help

Open an issue with the `help-needed` label if you're stuck.
