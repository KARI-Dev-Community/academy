# Project 08: Booking System

A booking and reservation management system.

## Learning Objectives

- Model complex booking data
- Handle date and time availability
- Implement email notifications
- Build calendar interfaces

## Tech Stack

- React (Vite)
- Node.js
- Express.js
- MongoDB
- Nodemailer

## Setup Guide

Frontend:

```bash
git clone https://github.com/kari-dev/project-08-booking-system.git
cd project-08-booking-system/frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
cp .env.example .env
npm run dev
```
